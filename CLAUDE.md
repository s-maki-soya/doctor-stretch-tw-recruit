# プロジェクト概要
ドクターストレッチ台湾の採用強化を目的とした採用LP。
コンテンツ言語は繁体字中国語（台湾向け）。
Claudeとの開発チャットは日本語で行うが、
サイト本体のテキスト・コピー・画像内文言はすべて繁体字中国語で作成する。
## コンバージョン導線（最重要）
LP → 公式LINE登録 → 応募 の一本道を徹底する。
- ファーストビューに公式LINEの友だち追加ボタンを配置
- 各セクションの末尾にもCTA（LINEボタン）を繰り返し配置
- LINEボタンはQRコード＋ボタンの両方を用意（PC/スマホ両対応）
- 応募フォームへの直接リンクは基本置かない。LINE経由に一本化する
- LINE公式アカウントのIDやリンクURLはダミーではなく指定された正式なものを使う（未確定の場合はプレースホルダーであることを明示）
## 技術構成
- Next.js 14.2.5（App Router、TypeScript）
- Tailwind CSS v3.4（カスタムブランドカラートークン設定済み）
- レスポンシブ対応必須、モバイルファースト（台湾の求職者はスマホ中心）
- 画像は `/public/images` に格納
- LINE公式アカウントURL: `https://lin.ee/s5IYRtF`（確定済み）
- デプロイ先: Vercel（GitHub連携）
## ディレクトリ構成
- `/app/page.tsx` - トップページ（LP本体、繁体字中国語）※EPERMロックあり、`page.new.tsx`に正版あり
- `/app/globals.css` - グローバルスタイル・Tailwindカスタムクラス
- `/components/` - 各セクションのReactコンポーネント
- `/public/images/` - 画像素材・LINE QRコード
- `tailwind.config.ts` - ブランドカラー定義（black:#111111, red:#D41F1F, lime:#C5E000）
## コンテンツ・言語のルール
- 本文・見出し・ボタン文言はすべて繁体字中国語で記述する
- 簡体字表記・簡体字由来の言い回しが混ざらないよう注意
- 台湾でよく使われる求人用語を使う（例：「應徵」「面試」「排班」「時薪/月薪」など、日本の求人用語の直訳を避ける）
- 給与・待遇の数字は最新の確定情報のみ記載
- 台湾の労働法・広告表記に抵触する誇大表現は避ける
- コード内のコメントや変数名は日本語または英語でよい（サイト表示文言のみ繁体字中国語）
## LINE導線まわりの実装メモ
- クリック計測（GA4やタグ）を入れる場合はLINEボタンのクリックをコンバージョンイベントとして計測できるようにする
- LINEボタンはページ内で常に視認できる位置（スクロール追従の固定ボタン等）を検討
## デザイン方針
- ブランドカラー・ロゴはドクターストレッチ公式ガイドラインに準拠
- 「安心感」「専門性」「成長できる環境」が伝わるトーンを意識
- 写真は現場スタッフの実写を優先（フリー素材感を避ける）
- LINEボタンは目立つ色・十分なタップ領域（モバイルでの誤タップ防止）
## SEO・流入対策
- タイトルタグ・メタディスクリプションは繁体字中国語で、「台灣 招募」「整復推拿 徵才」等の関連語を意識
- OGP画像を設定（SNSシェア時の見栄え、LINEシェアでの表示も確認）
- 画像は圧縮し表示速度を担保
## やってはいけないこと
- 給与・待遇について未確定の情報を記載しない
- LINE ID・URLを確認なしに架空のもので確定版として扱わない
- 応募フォームへの直接導線を勝手に追加しない（LINE一本化の導線を崩さない）
- 他社採用サイトの文章・デザインをそのまま流用しない

## PC操作に関するルール（重要）
ClaudeがこのPCを直接操作（クリック・入力・ファイル操作等）する場合は、**必ず事前にユーザーの許可を得ること。**
無断でPC操作を開始してはならない。

### 操作が必要なタスクの提示方法
PC操作が必要な場面では、以下の2パターンを必ず提示し、ユーザーに選択してもらう。

**パターンA：Claudeが操作する**
ClaudeがPC画面を見ながらマウス・キーボードを自動操作する。
ユーザーはPCから手を離して待つだけでよい。
→ 選択した場合のみ操作を開始する。

**パターンB：自分で操作する**
Claudeは手順を日本語でステップバイステップに説明する。
ユーザー自身がその手順に従って操作する。
→ Claudeは操作せず、サポート役に徹する。

### 提示例
```
次のステップでPC操作が必要です。
どちらで進めますか？

A）Claudeが操作する（自動）
B）手順を教えてもらって自分で操作する
```
